Movie Theater Data

I started using MoviePass a little while ago, so I've been going to the theater a lot. With the service falling apart, I thought it would be fun to take a look at the data on the movies I saw.

The data is visualized with D3. All the data is pulled directly from a Google Sheet with Tabletop.js on page load. Some of the data comes from OMDb. I referenced these projects for the calendar chart and this project for the stacked bar chart. I used this stylesheet for the Solarized Dark color scheme.
